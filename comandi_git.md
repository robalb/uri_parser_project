# qui ci sono i comandi git utili

Per lo stato: git status
Per aggiungere un file al "commit" : git add nome_file (oppure git add . per tutti i file)
Per salvare il "commit" : git commit -m "descrizione del commit"
Per salvare il "commit" sul server: git push origin master (dove master è il nome del commit del quale si desidera fare push sul server origin)
Per creare un "commit" con un nome: git checkout -b nome-commit (questo comando si può usare per spostarsi al "commit" con il nome passato al comando, se non esiste viene creato)
